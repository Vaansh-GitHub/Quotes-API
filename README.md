# How it Works ?

<p>Step 1: Just fork the Quotes API project and download it on your system</p>
<p>Step 2: Then simply run <code>npm i</code> to install the required dependencies for functioning of the api</p>
<br>
<li>Make <b>GET</b> request to <code>/quotes</code> to get a random quote from a large set of quotes</li>
<li>Make <b>POST</b> request to <code>/quotes</code> with the author name to be passed in the "body" to get the quotes of the given author</li>
